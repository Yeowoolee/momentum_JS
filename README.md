# momentum_JS
1. 현재시간을 표시    
2. ToDo list등록/삭제    
3. 현 위치와 기온 표시(openweather API 사용)    
4. 사용자, ToDo list 정보를 브라우저 로컬 저장소에 저장
5. 랜덤으로 변경되는 배경화면

>사용언어 vanilla JS, html, css
